# Logo
My version of LOGO with Animation

Run Here: http://htmlpreview.github.io/?https://github.com/imlolman/Logo/blob/master/index.html

# Commands Supported

rt - right turn
lt - left turn
fd - forward
bk - backward
pu - pen up
pd - pen down
ht - hide turtle
st - show turtle
repeat - to repeat the commands.

# Repeat syntax

repeat loopCount [ commands ]
